Atmos

The ATMOS app is a green app designed to add simplicity for Electric Vehicle Owners, as well as supports and entices others to the campaign for renewable energy!
ev_station

Motivation

  • 2017 was hottest year on record
  • Sea levels are rising
  • number of wildfires is increasing
  • Dangerous heat waves expanding
  • Increase in severe droughts
  • Extreme storms happening in unprecedented numbers
  • Enterprise only.

Technology

  • New Technology: Materialize css framework
  • Firebase - used for storing contact information and user-input from our app
  • All Stations API - used by grabbing user-input and finding the closest electric charging stations to their location
  • Google Maps API - used for creating markers on the map for each station and storing the addresses in the "infowindow" which displays upon clicking a marker
  • Geolocation (Coming Soon) - used for finding your location and providing directions between you and the charging stations
  • Google Maps App (Coming Soon) - when you click a charging station on your phone, the Google Maps Mobile App will open

Design

  • Implementation of Ui/Ux using materialize
  • Implementation of all major components in the materialize documentation
  • Downloaded a custom materialize theme as a foundation to research the inner workings of a css framework
  • Researched how css frameworks are built
